This portfolio is built for speed and simplicity. Unlike standard templates that lock content into complex TypeScript arrays, this version has been **refactored to be 100% Markdown-driven**.

### Key Features

- **Pure Markdown Content:** Adding a new project is as simple as dropping a `.md` file into a folder.
- **High Performance:** Built with **Astro**, meaning zero JavaScript is sent to the client by default.
- **Type Safety:** Uses Astro Content Collections to ensure all project metadata (dates, tags, images) is correct during the build.
- **Clean Design:** A terminal-inspired, minimalist aesthetic focusing on readability.
